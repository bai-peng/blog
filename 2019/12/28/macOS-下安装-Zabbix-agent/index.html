macOS 下安装 Zabbix agent

Posted on 2019-12-28
1.下载

主页地址：https://www.zabbix.com/download_agents

cd /tmp curl -O https://assets.zabbix.com/downloads/4.0.16/zabbix_agent-4.0.16-macos-amd64-gnutls.pkg sudo installer -pkg zabbix_agent-4.0.16-macos-amd64-gnutls.pkg -target /

sudo vi /usr/local/etc/zabbix/zabbix_agentd.conf

2.修改配置文件

Server={SERVERIP} ServerActive={SERVERIP} Hostname={hostname}

3.开启/停止参数

sudo launchctl start com.zabbix.zabbix_agentd sudo launchctl stop com.zabbix.zabbix_agentd

注：主机执行 active checks 时， Zabbix agent 需要定义主机名字。 而且, agent端的主机名字必须和前端配置的主机名字 “Host name“完全匹配。

4.Zabbix server 添加 zabbix agent

1.点击 configuration - Hosts - “Create host”

Groups:当前环境定义的组
Host name 填写 上面 /usr/local/etc/zabbix/zabbix_agentd.conf 中配置的”Hostname“
Agent interfaces ：上文中zabbix agent 的IP/DNS名称，注意这里 Connect to的选择；
Templates： “Template OS Mac OS X”
添加成功后如下；
