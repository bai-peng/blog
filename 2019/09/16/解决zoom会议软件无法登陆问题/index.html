解决zoom.us 会议软件无法登陆问题

macos 下解决zoom无法使用的问题，编辑本机hosts文件即可解决。

操作步骤: 打开terminal，粘贴以下内容,然后重新打开zoom就可以连接了，等待zoom服务恢复正常后删除下面的内容就可以了。

Windows 系统一样，将以上内容通过管理员权限使用记事本打开 C:\Windows\System32\drivers\etc\hosts 文件 粘贴以上内容保存即可。

Windows 操作方式 ，

点击Windows 开始按钮,输入 notepad ，然后右键在记图标上面右键点击 “以管理员身份打开”,

打开后的记事本中点击 菜单-打开命令，在文件名一栏输入
“C:\Windows\System32\drivers\etc\hosts” 然后点击打开将以上代码区的文件粘贴到尾部就可以解决zoom无法连接的问题了。
